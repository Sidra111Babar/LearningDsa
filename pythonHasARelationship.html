Has A Relationship

  • Association
  • Aggregation
  • Composition

Association

Association represents a general relationship between two classes. It shows that one class uses or interacts with another class. Weakest form of relationship. A teacher and a student can interact, but neither owns the other.


class Teacher:
    def __init__(self, name):
        self.name = name

    def teach(self):
        print(f"{self.name} is teaching.")

class Student:
    def __init__(self, name):
        self.name = name

    def study(self):
        print(f"{self.name} is studying.")

# Association: Teacher interacts with Student
teacher = Teacher("Mr. Smith")
student = Student("Alice")

teacher.teach()
student.study()

Aggregation

Aggregation is a form of association with a "whole-part" relationship. The part (child) can exist independently of the whole (parent).A weak form of ownership.


class Department:
    def __init__(self, name):
        self.name = name

class University:
    def __init__(self, name):
        self.name = name
        self.departments = []  # Aggregates departments

    def add_department(self, department):
        self.departments.append(department)

# Creating departments
dept1 = Department("Computer Science")
dept2 = Department("Physics")

# Creating a university and adding departments
university = University("Tech University")
university.add_department(dept1)
university.add_department(dept2)

print(f"{university.name} has the following departments:")
for dept in university.departments:
    print(dept.name)

Composition

Composition is a strong form of aggregation where the "whole" is responsible for the lifecycle of its parts. If the whole is destroyed, the parts are also destroyed.It is a Strong ownership


class Engine:
    def __init__(self, type):
        self.type = type

    def start(self):
        print(f"{self.type} engine is starting.")

class Car:
    def __init__(self, brand, engine_type):
        self.brand = brand
        self.engine = Engine(engine_type)  # Composition: Engine is part of Car

    def start_car(self):
        print(f"Starting {self.brand} car...")
        self.engine.start()

# Creating a car with an engine
car = Car("Tesla", "Electric")
car.start_car()
