Strings

A string in Python is a sequence of characters enclosed in single ('), double ("), or triple quotes (''' or """`). Strings are one of the most commonly used data types and are immutable, meaning they cannot be changed once created.

Example


    # Single and Double quote
    string1 = 'Hello, World!'
    string2 = "Python Strings"
    print(string1)  # Output: Hello, World!
    print(string2)  # Output: Python Strings

Example


    # Single and Double quote
    multiline_string = '''This is a multiline string.
    It can span multiple lines.'''
    print(multiline_string)

Common String Operations

  1. Accessing Characters: Strings are indexed, allowing you to access individual characters 
    
    string = "Python"
    print(string[0])  # Output: P
    print(string[-1]) # Output: n
  2. String Slicing: Extract a part of the string using slicing: 
    
    string = "Programming"
    print(string[0:4])   # Output: Prog
    print(string[-3:])   # Output: ing
  3. String Length: Use the len() function 
    
    string = "Hello"
    print(len(string))  # Output: 5
  4. Split a String: 
    
    text = "Python is fun"
    print(text.split())  # Output: ['Python', 'is', 'fun']
  5. String Formatting or f String: 
    
    name = "Sidra"
    fname = "Babar" 
    city = "Dreams"
    print(f"My name is {name} and my father name is {fname}.I lives in {city}")

# ------------------- Python String Methods Demonstration -------------------

# Case Conversion
str1 = "hello World"
print(str1.capitalize())  # Capitalizes the first character
print(str1.lower())       # Converts to lowercase
print(str1.upper())       # Converts to uppercase
print(str1.swapcase())    # Swaps case
print(str1.title())       # Capitalizes the first character of each word

# Alignment
str2 = "Python"
print(str2.center(10, '-'))  # Centers the string with padding
print(str2.ljust(10, '-'))   # Left-justifies the string with padding
print(str2.rjust(10, '-'))   # Right-justifies the string with padding
print("42".zfill(5))         # Pads the string with zeros on the left

# Searching
str3 = "hello"
print(str3.count('l'))       # Counts occurrences of a substring
print(str3.find('e'))        # Finds the first occurrence of a substring
print(str3.rfind('l'))       # Finds the last occurrence of a substring
print(str3.index('e'))       # Finds the index of a substring
print(str3.rindex('l'))      # Finds the last index of a substring

# Checking
str4 = "Hello123"
print(str4.startswith('He'))   # Checks if the string starts with a prefix
print(str4.endswith('23'))     # Checks if the string ends with a suffix
print(str4.isalnum())          # Checks if all characters are alphanumeric
print(str4.isalpha())          # Checks if all characters are alphabetic
print(str4.isdigit())          # Checks if all characters are digits
print("  ".isspace())          # Checks if all characters are whitespace

# Modifying
str5 = "   Python   "
print(str5.strip())            # Removes leading and trailing whitespace
print(str5.rstrip())           # Removes trailing whitespace
print(str5.lstrip())           # Removes leading whitespace

str6 = "Python is awesome!"
print(str6.replace("awesome", "great"))  # Replaces a substring with another

# Splitting and Joining
str7 = "apple,banana,cherry"
print(str7.split(','))       # Splits the string at each occurrence of a separator
print(" ".join(["I", "love", "Python"]))  # Joins elements of a list with a separator

# Other
