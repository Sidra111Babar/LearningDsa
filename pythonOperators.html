Operators

  • Arithmetic Operators

    Arithmetic operators perform mathematical operations.

    Operator Description Example
    + Addition 5 + 3 = 8
    - Subtraction 5 - 3 = 2
    * Multiplication 5 * 3 = 15
    / Division 5 / 2 = 2.5
    // Floor Division 5 // 2 = 2
    % Modulus 5 % 2 = 1
    ** Exponentiation 5 ** 2 = 25

    Example

    
    a, b = 5, 2
    print(a + b)   # 7
    print(a ** b)  # 25

  • Assignment Operators

    Assignment operators assign values to variables.

    Operator Description Example
    = Assigns value a = 5
    += Add and assign a += 3
    -= Subtract and assign a -= 2
    *= Multiply and assign a *= 2
    /= Divide and assign a /= 2
    //= Floor divide and assign a //= 2
    %= Modulus and assign 5 %= 2
    **= Exponentiation and assign a **= 2

    Example

    
    a = 5
    a += 2  # a = a + 2
    print(a)  # 7

  • Comparison Operators

    Comparison operators compare two values and return a boolean.

    Operator Description Example
    == Equal to 5 == 5 → True
    != Not equal to 5 != 3 → True
    > Greater than 5 > 3 → True
    < Less than 3 < 5 → True
    >= Greater than or equal to 5 >= 3 → True
    <= Less than or equal to 3 <= 5 → True

    Example

    
    a, b = 5, 3
    print(a > b)  # True
    print(a == b)  # False

  • Logical Operators

    Logical operators are used to combine conditional statements.

    Operator Description Example
    and True if both are True 5 > 3 and 5 < 10 → True
    or True if at least one is True 5 > 3 or 5 > 10 → True
    not Negates the result not(5 > 3) → False

    Example

    
    a, b = True, False
    print(a and b)  # False
    print(a or b)   # True

  • Identity Operators

    Identity operators check if two objects are the same.

    Operator Description Example
    is True if objects are same a is b
    is not True if objects are not same a is not b

    Example

    
     a = [1, 2, 3]
     b = a
     c = [1, 2, 3]
     print(a is b)       # True
     print(a is c)       # False (different objects)

  • Membership Operators

    Membership operators test for membership in a sequence (e.g., lists, strings).

    Operator Description Example
    in True if element is in sequence 'a' in 'apple' → True
    not in True if element not in sequence 'b' not in 'apple' → True

    Example

    
    a = [1, 2, 3]
    print(2 in a)       # True
    print(4 not in a)   # True

  • Bitwise Operators

    Bitwise operators are used to perform operations on binary numbers at the bit level. They treat numbers as sequences of binary digits (bits) and operate on each bit individually.

    Operator Description Example
    & Bitwise AND 5 & 3 = 1
    ` ` Bitwise OR
    ^ Bitwise XOR 5 ^ 3 = 6
    ~ Bitwise NOT ~5 = -6
    << Left shift 5 << 1 = 10
    >> Right shift 5 >> 1 = 2

    Example

    
    a, b = 5, 3
    print(a & b)  # 1
    print(a | b)  # 7
    print(a << 1) # 10

    • Bitwise AND (&)

      Compares each bit of two numbers.The result is 1 only if both corresponding bits are 1; otherwise, it's 0.

      Example

      
    a = 5      # Binary: 0101
    b = 3      # Binary: 0011
    result = a & b  # Binary: 0001 → Decimal: 1
    print(result)  # Output: 1

    • Bitwise OR (|)

      Compares each bit of two numbers.The result is 1 if either of the corresponding bits is 1.

      Example

      
    a = 5      # Binary: 0101
    b = 3      # Binary: 0011
    result = a | b  # Binary: 0111 → Decimal: 7
    print(result)  # Output: 7

    • Bitwise XOR (^)

      Compares each bit of two numbers.The result is 1 if the corresponding bits are different; otherwise, it's 0.

      Example

      
    a = 5      # Binary: 0101
    b = 3      # Binary: 0011
    result = a ^ b  # Binary: 0110 → Decimal: 6
    print(result)  # Output: 6

    • Bitwise NOT (~)

      Inverts all the bits of a number.Changes 1 to 0 and 0 to 1.The result is the two's complement of the number, calculated as -(n + 1).

      Note:Negative results occur because Python uses signed integers, where the most significant bit represents the sign.

      Example

      
    a = 5      # Binary: 0101
    result = ~a  # Binary: 1010 → Decimal: -6
    print(result)  # Output: -6

    • Left Shift (<<)

      Shifts the bits of a number to the left by the specified number of positions.Adds 0s to the right and effectively multiplies the number by 2^n (where n is the shift count).

      Example

      
    a = 5      # Binary: 0101
    result = a << 1  # Binary: 1010 → Decimal: 10
    print(result)  # Output: 10

    • Right Shift (>>)

      Shifts the bits of a number to the right by the specified number of positions.Discards bits shifted off and fills the leftmost bits with 0s (for positive numbers).

      Example

      
    a = 5      # Binary: 0101
    result = a >> 1  # Binary: 0010 → Decimal: 2
    print(result)  # Output: 2

    • Combined Bitwise

      Bitwise operators can be combined for complex operations.

      
    a = 5       # Binary: 0101
    b = 3       # Binary: 0011
    result = (a & b) | (a ^ b)  # (0001) | (0110) = 0111 → Decimal: 7
    print(result)  # Output: 7
      Use Cases
    • Low-Level Programming: Bitwise operators are often used in hardware-level programming to control or manipulate specific bits.
    • Optimized Calculations: Left shifts (<<) are faster than multiplying by powers of 2.Right shifts (>>) are faster than dividing by powers of 2.
    • Flags and Masks: Used to set, clear, or check specific bits in a value.
