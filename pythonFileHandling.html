File Handling

File handling in Python allows you to work with files for reading, writing, or appending data. It provides built-in functions and methods for file operations, making it easy to manipulate text and binary files.

Methods

Mode Description
"r" Read-only (default mode).
"w" Write-only (overwrites existing files).
"a" Append mode. Adds to the file.
"b" Binary mode (used with other modes).
"x" Exclusive creation (fails if file exists).
"t" Text mode (default).
"r+" Read and write.
"w+" Write and read (overwrites existing).
"a+" Append and read.

"r" (Read-Only)

Opens a file for reading. File must exist.

Copied!

Not Copied!


with open("example.txt", "r") as f:
    content = f.read()
    print(content)

"w" (Write-Only)

Opens a file for writing. Overwrites existing content or creates a new file.

Copied!

Not Copied!


with open("example.txt", "w") as f:
    f.write("This is a new file content.")

"a" (Append Mode)

Opens a file for appending. Content is added to the end of the file.

Copied!

Not Copied!


with open("example.txt", "a") as f:
    f.write("\nAppending new content.")

"b" (Binary Mode)

Used for reading or writing binary data like images.

Copied!

Not Copied!


with open("image.png", "rb") as f:
    binary_data = f.read()
    print(binary_data)
with open("copy_image.png", "wb") as f:
    f.write(binary_data)  # Writes previously read binary data

"x" (Exclusive Creation)

Creates a new file. Fails if the file already exists.

Copied!

Not Copied!


try:
    with open("newfile.txt", "x") as f:
        f.write("This file is created using 'x' mode.")
except FileExistsError:
    print("File already exists.")

"t" (Text Mode)

Default mode. Used for text data (often combined with "r", "w", etc.).

Copied!

Not Copied!


with open("example.txt", "rt") as f:  # "t" is implied
    print(f.read())

"r+" (Read and Write)

Opens a file for both reading and writing. File must exist.

Copied!

Not Copied!


with open("example.txt", "r+") as f:
    content = f.read()
    print("Before:", content)
    f.seek(0)  # Move to the beginning of the file
    f.write("Updated content.")

"w+" (Write and Read)

Opens a file for writing and reading. Overwrites existing content.

Not Copied!


with open("example.txt", "w+") as f:
    f.write("New content for the file.")
    f.seek(0)  # Move to the beginning to read
    print(f.read())

"a+" (Append and Read)

Opens a file for appending and reading.

Not Copied!


with open("example.txt", "a+") as f:
    f.write("\nAdding more content.")
    f.seek(0)  # Move to the beginning to read the entire file
    print(f.read())
