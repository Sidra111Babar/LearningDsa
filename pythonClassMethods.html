Class Methods

In Python, staticmethod, classmethod, and property are all ways to define methods with different behaviors in a class.

Difference between cls and self in python?

Static Method

A static method is a method that belongs to the class but does not take a reference to the instance (self) or the class (cls) as its first parameter. It behaves like a regular function that just happens to be defined inside the class. It cannot modify the state of the instance or class, but it can still be used to group functions that logically belong to the class.

Not Copied!


class MyClass:
    @staticmethod
    def static_method(x, y):
        return x + y

# Usage
print(MyClass.static_method(5, 10))  # Output: 15

Class Method

A class method takes a reference to the class (cls) as its first parameter instead of an instance (self). Class methods can modify the class state (e.g., class variables) and are often used for factory methods or methods that work with class-level data.

Copied!

Not Copied!


class Company:
    company = "Google"
    def companyName(self):
        print(f"Name of the company is {self.company}")
    def changeCompanyName(self,newCompany):
        self.company = newCompany # Method without @classmethod take argument as self(Instance) 
    @classmethod     # Now it takes argument as cls(Directly edit class) no instance
    def changeCompanyNameUsingClassMethod(cls,newCompany):
        cls.company = newCompany
obj= Company()
obj.companyName()
obj.changeCompanyName("Microsoft")
obj.companyName()
# Now check that company name is actually change or not
print(Company.company)    # output is google no microsoft so there is need to define this function with @classmethod that's actually work
obj.changeCompanyNameUsingClassMethod("Microsoft")
print(Company.company)     # Now it actually change

Change value of class variable directly

class MyClass:
    class_variable = 0
    
    @classmethod
    def class_method(cls):
        cls.class_variable += 1
        print(f"Class variable is now: {cls.class_variable}")

# Usage
MyClass.class_method()  # Output: Class variable is now: 1
MyClass.class_method()  # Output: Class variable is now: 2

Property

A property allows you to define a method that behaves like an attribute. The @property decorator allows you to define a getter, and you can also use @.setter to define a setter. This provides a way to manage access to instance attributes without directly accessing them.

Copied!

Not Copied!


class MyClass:
    def __init__(self, value):
        self._value = value
    
    @property
    def value(self):
        return self._value
    
    @value.setter
    def value(self, new_value):
        if new_value >= 0:
            self._value = new_value
        else:
            print("Value must be non-negative")

# Usage
obj = MyClass(10)
print(obj.value)  # Output: 10
obj.value = 20
print(obj.value)  # Output: 20
obj.value = -5    # Output: Value must be non-negative
