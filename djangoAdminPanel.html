Django Admin Panel

Django Admin is a built-in web-based interface provided by Django for managing database records in a user-friendly way. It allows developers and administrators to create, update, delete, and view data without writing SQL queries.

How to Setup

  • Same as Django setup
  • After Setting up a project and after adding an app make models in models.py
  • Make models in models.py as in the code below

    • Copied!

    Not Copied!

    
from django.db import models

class Customer(models.Model):
    first_name = models.CharField(max_length=50)
    last_name = models.CharField(max_length=50)
    email = models.EmailField(unique=True)
    phone = models.CharField(max_length=15, blank=True, null=True)
    address = models.TextField(blank=True, null=True)
    created_at = models.DateTimeField(auto_now_add=True)
    
class Order(models.Model):
    customer = models.ForeignKey(Customer, on_delete=models.CASCADE, related_name="orders")
    order_date = models.DateTimeField(auto_now_add=True)
    total_amount = models.DecimalField(max_digits=10, decimal_places=2)
    status = models.CharField(
    max_length=50,
    choices=(
        ("Pending", "Pending"),
        ("Shipped", "Shipped"),
        ("Delivered", "Delivered"),
        ("Cancelled", "Cancelled"),
    ),
    default="Pending"  # Default value set to "Pending"
)

list_display

The list_display attribute allows you to control which fields appear in the Django admin list view.

inlines

Django inlines allow displaying related models within a parent model’s admin page.

Copied!

Not Copied!


class OrderInline(admin.StackedInline):  # StackedInline for vertical layout
    model = Order
    extra = 0

@admin.register(Customer)
class CustomerAdmin(admin.ModelAdmin):
    list_display = ("first_name", "last_name", "email", "phone", "address", "created_at")
    search_fields = ("first_name", "last_name", "email", "phone")
    inlines = [OrderInline]  # This shows related orders inside the Customer admin panel

StackedInline shows inline forms in a vertical layout.

If you want a tabular (horizontal) layout, use admin.TabularInline.

mark_as_shipped.short_description = "Mark selected orders as Shipped"

forms.ModelForm.

You can modify Django’s default admin forms using this.

Copied!

Not Copied!


from django import forms

class CustomerForm(forms.ModelForm):
    class Meta:
        model = Customer
        fields = '__all__'
        widgets = {
            'address': forms.Textarea(attrs={'rows': 30})
        }

Here, we’ve customized the address field to display a larger textarea.
