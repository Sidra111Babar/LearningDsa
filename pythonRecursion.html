Recursion

Recursion in Python is a programming technique where a function calls itself to solve a problem. Each recursive call works on a smaller or simpler instance of the problem, with the goal of reaching a base case that stops the recursion.

  • Base Case:

    The condition under which the recursion stops.Without a base case, the recursion would continue indefinitely, leading to a stack overflow.

  • Recursive Case:

    The part of the function where the problem is divided into smaller subproblems, and the function calls itself.

Working Scenerio

Each recursive call adds a new layer to the call stack (a data structure that tracks active function calls). Once the base case is reached, the recursion starts "unwinding," returning results back through the stack.

def factorial(n):
    # Base case
    if n == 0 or n == 1:
        return 1
    # Recursive case
    return n * factorial(n - 1)

# Example usage
print(factorial(5))  # Output: 120
