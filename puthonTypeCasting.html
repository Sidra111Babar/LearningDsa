TypeCasting

Type casting in Python refers to the process of converting one data type into another. Python provides both implicit and explicit type casting methods.

    Types

  • Implicit Type Casting

    Definition: Python automatically converts a variable from one type to another without requiring the programmer's intervention.

    Characteristics: It is safe and does not result in data loss.

    Real-World Scenario: When calculating the total price in an e-commerce application, the number of items (integer) might be multiplied by the price per item (float), and Python will handle the type conversion automatically.

    Precedence Order: The precedence order of data types in Python refers to how Python handles type conversions when multiple data types are involved in operations. Generally, the order of precedence is

    complex > float > int > bool

    Example 1

    
    # integer and float                                    
    num = 10    # Integer
    result = num + 2.5  # Float
    print(result)       # Output: 12.5
    print(type(result)) # Output:<class 'float'>

    Example 2

    
    # integer and complex
    a = 3.5  # float
    b = 2 + 4j  # complex
    result = a + b
    print(result)       # Output: (5.5+4j)
    print(type(result)) # Output: <class 'complex'>

    Example 3

    
    # integer and boolean
    x = 10  # int
    y = True  # bool (treated as 1)
    result = x + y
    print(result)       # Output: 11
    print(type(result)) # Output: <class 'int'>

    Example 4

    
    # Mixed type
    x = 10  # int
    y = 3.5  # float
    z = 2 + 4j  # complex
    result = x + y + z
    print(result)       # Output: (15.5+4j)
    print(type(result)) # Output: <class 'complex'>

  • Explicit Type Casting

    Definition: The programmer manually converts a variable from one type to another using Python's built-in functions like int(), float(), str(), list(), etc

    Characteristics: Needs to be explicitly specified by the programmer and might result in data loss or errors if not handled correctly.

    Example 

    
    # Convert float to int
    price = 25.99
    rounded_price = int(price)
    print(rounded_price)  # Output: 25
    
    # Convert string to int
    quantity = "5"
    total = int(quantity) * 10
    print(total)  # Output: 50

    Real-World Scenario 1: In a banking application, user input from forms is usually received as a string. To perform calculations like adding balance or interest, the strings must be converted into integers or floats.

    Example 

    
    deposit = input("Enter deposit amount: ")  # Input: "1000"
    total_balance = int(deposit) + 500
    print(total_balance)  # Output: 1500

    Real-World Scenario 2: Parsing data from a JSON or CSV file might require converting strings to numeric types for analysis or visualization.
