Loops

In Python, loops are used to execute a block of code repeatedly as long as a certain condition is met. Python provides two main types of loops

  • for Loop

    The for loop is used to iterate over a sequence (like a list, tuple, dictionary, string, or range). It is useful when the number of iterations is known beforehand.

    
# syntax
for variable in sequence:
# code block                                
                                
    
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)
    # output
    # apple
    # banana
    # cherry          
                                
    
for i in range(5):
print(i)    
# output  
# 0
# 1
# 2
# 3
# 4                          
                                
    
for char in "Python":
print(char)   
# output
# P
# y
# t
# h
# o
# n 

                                
    
for i in range(10):
if i == 5:
    break
print(i)       # output is until 4                                                       
                                
    
for i in range(3):
    if i == 1:
        pass  # Placeholder, does nothing
    print(i)         # 0 1 2                                                     
                                
    
for i in range(5):
    if i == 2:
        continue
    print(i)      # output is from 0 1 3 4

  • while Loop

    The while loop is used to repeat a block of code as long as the given condition is True. It is useful when the number of iterations is not known beforehand.

    
# syntax
while condition:
    # code block 
                                
    
count = 0
while count < 5:
    print(count)
    count += 1
    # output 
# 0
# 1
# 2
# 3
# 4

Nested Loop

A loop inside another loop is called a nested loop.


for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
        # output 
# i=0, j=0
# i=0, j=1
# i=1, j=0
# i=1, j=1
# i=2, j=0
# i=2, j=1

Infinite Loop

An infinite loop runs forever if the condition never becomes False. Use break to exit such loops.


while True:
    response = input("Type 'exit' to stop: ")
    if response == 'exit':
        break
