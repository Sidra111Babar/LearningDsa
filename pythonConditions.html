Conditions

In Python, conditions are used to execute specific blocks of code based on whether a particular condition evaluates to True or False. Conditions are implemented using the if, elif, and else statements.

  • if Statement

    The if statement is used to check a condition. If the condition evaluates to True, the block of code inside the if statement is executed.

    
if condition:
# Code to execute if the condition is True 
                                
    
age = int(input("Enter your age: "))
if age >= 18:
print("You are eligible to vote.")

  • if-else Statement

    The else block is used to execute code when the if condition evaluates to False.

    
if condition:
    # Code to execute if the condition is True
else:
    # Code to execute if the condition is False                                    
    
age = int(input("Enter your age: "))
if age >= 18:
    print("You can vote.")
else:
    print("You cannot vote.")

  • if-elif-else Statement

    The elif (short for "else if") statement allows you to check multiple conditions sequentially. The first condition that evaluates to True will execute, and the rest will be ignored.

    
if condition1:
    # Code to execute if condition1 is True
elif condition2:
    # Code to execute if condition2 is True
else:
    # Code to execute if none of the above conditions are True
                                
    
num = int(input("Enter a number: "))

if num < 0:
    print("The number is negative.")
elif num == 0:
    print("The number is zero.")
else:
    print("The number is positive.")

  • Nested if Statements

    You can place an if statement inside another if statement to create nested conditions.

    
if condition1:
    if condition2:
        # Code to execute if both condition1 and condition2 are True
    else:
        # Code to execute if condition1 is True but condition2 is False
else:
    # Code to execute if condition1 is False
                            
    
age = int(input("Enter your age: "))

if age > 0:
    if age < 18:
        print("You are underage.")
    else:
        print("You are an adult.")
else:
    print("Invalid age entered.")

  • Multiple if

    In this scenario, multiple independent if conditions are checked. Each condition is evaluated separately, meaning that more than one condition can be True, and all corresponding blocks of code will execute.

    
number = int(input("Enter a number: "))
if number % 2 == 0:
    print(f"{number} is even.")

if number > 0:
    print(f"{number} is positive.")

if number % 5 == 0:
    print(f"{number} is divisible by 5.")

if number > 100:
    print(f"{number} is greater than 100.")

Example


marks = int(input("Enter your marks: "))

if marks >= 90:
grade = 'A'
elif 80 <= marks < 90:
grade = 'B'
elif 70 <= marks < 80:
grade = 'C'
elif 60 <= marks < 70:
grade = 'D'
else:
grade = 'F'

if grade == 'F':
print("Fail. Better luck next time!")
else:
print(f"You passed with grade {grade}.")

    Cautions

  • Readability: Use proper indentation (4 spaces or a tab per level).Avoid deeply nested conditions; refactor using logical operators if possible.
  • Logical Operators: Combine multiple conditions using and, or, and not for cleaner code. 
    
if 10 <= num <= 20:
print("Number is between 10 and 20.")
  • Avoid Unnecessary elif and else: If the logic can be handled with if statements alone, avoid elif or else. 
    
if num < 0:
print("Negative")
if num == 0:
print("Zero")
  • Handle Edge Cases: Always account for boundary values or unexpected input (e.g., negative numbers, strings, etc.).
