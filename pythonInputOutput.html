Input and Output

Python provides simple ways to interact with the user through input (data provided by the user) and output (data shown to the user).The input() function is used to take input from the user during program execution.

Note: The input() function always returns input as a string.Convert it to the desired data type using typecasting if needed.

Example


deposit = input("Enter deposit amount: ")  # Input: "1000"
total_balance = int(deposit) + 500
print(total_balance)  # Output: 1500
                        

# Taking string input
name = input("Enter your name: ")
print("Hello, " + name)

# Taking integer input
age = int(input("Enter your age: "))
print("You are", age, "years old.")

# Taking float input
height = float(input("Enter your height in meters: "))
print("Your height is", height, "meters.")                            
                        

# Taking multiple inputs using split()
x, y = input("Enter two numbers separated by space: ").split()
print("First number:", x)
print("Second number:", y)

# Converting inputs to integers
x, y = map(int, input("Enter two numbers separated by space: ").split())
print("Sum:", x + y)
