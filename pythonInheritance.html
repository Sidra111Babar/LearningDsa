Inheritance

Inheritance is a feature in object-oriented programming (OOP) that allows a class (child class) to inherit attributes and methods from another class (parent class). This promotes code reuse and enables the creation of hierarchical class structures.

Explanation

  • Parent Class (Animal): Defines shared attributes and methods.
  • Child Classes (Dog, Cat): Inherit from the Animal class and override the speak method to provide specific behavior.
  • Objects (dog, cat): Created from child classes, inheriting the name attribute and having specific implementations of the speak method.

Note:

Use inheritance when a "is-a" relationship exists between classes (e.g., a Dog is an Animal).Prefer composition over inheritance if the relationship is more about "has-a" or "uses" rather than "is-a."

Advantages of Inheritance

  1. Code Reusability: Reduces duplication by reusing methods and attributes from the parent class.
  2. Extensibility: Allows easy addition of new features without modifying existing code.
  3. Polymorphism: Enables child classes to provide their implementations of methods, enhancing flexibility.
  4. Improved Readability: Provides a clear hierarchy, making the codebase easier to understand.

Use Cases

  1. Hierarchical ModelsExample: Creating a hierarchy of vehicles like Car, Truck, and Bike, all inheriting from a Vehicle class.
  2. Shared Functionality: Example: A User class for common attributes, and Admin and Customer classes for role-specific functionality.
  3. Framework Design: Example: Web frameworks like Django use inheritance to define views, models, and forms.
  4. Overriding and Extending Behavior: Example: Subclassing a built-in Python class like list or dict to add custom methods.

# Parent class
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        return f"{self.name} makes a sound."

# Child class inheriting from Animal
class Dog(Animal):
    def speak(self):
        return f"{self.name} barks."

# Child class inheriting from Animal
class Cat(Animal):
    def speak(self):
        return f"{self.name} meows."

# Instantiating objects
dog = Dog("Buddy")
cat = Cat("Whiskers")

print(dog.speak())  # Output: Buddy barks.
print(cat.speak())  # Output: Whiskers meows.

Types Of Inheritance

Python supports five types of inheritance

Single Inheritance

A child class inherits from a single parent class.

Example

Copied!

Not Copied!


class Parent:
    def show(self):
        return "This is a parent class."

class Child(Parent):
    def display(self):
        return "This is a child class."

obj = Child()
print(obj.show())    # Output: This is a parent class.
print(obj.display()) # Output: This is a child class.

Multiple Inheritance

A child class inherits from more than one parent class.

Example

class Parent1:
    def feature1(self):
        return "Feature from Parent1."

class Parent2:
    def feature2(self):
        return "Feature from Parent2."

class Child(Parent1, Parent2):
    def display(self):
        return "This is a child class."

obj = Child()
print(obj.feature1()) # Output: Feature from Parent1.
print(obj.feature2()) # Output: Feature from Parent2.

Multilevel Inheritance

A child class inherits from a parent class, and another child class inherits from that child class.

class Grandparent:

class Parent(Grandparent):
    def feature2(self):
        return "Feature from Parent."

class Child(Parent):
    def feature3(self):
        return "Feature from Child."

obj = Child()
print(obj.feature1()) # Output: Feature from Grandparent.
print(obj.feature2()) # Output: Feature from Parent.
print(obj.feature3()) # Output: Feature from Child.

Heirarchical Inheritance

Multiple child classes inherit from the same parent class.

    def display2(self):
        return "This is Child2."

obj1 = Child1()
obj2 = Child2()
print(obj1.feature()) # Output: Feature from Parent.
print(obj2.feature()) # Output: Feature from Parent.

Hybrid Inheritance

A combination of multiple types of inheritance.
